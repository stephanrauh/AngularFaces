This demo is about testing BabbageFaces. To see the effect open the web developer tool which is activated by hitting the F12 key in most browsers. Alternatively you can use a tool like Fiddler or Paros to intercept the HTML requests. In any case you should see AJAX responses consisting of <delete>, <insert> and <change> after clicking a check box or hitting the button. Without BabbageFaces responses consist of %lt;update> statements only, thus requiring more bandwith and more browser rendering time.
