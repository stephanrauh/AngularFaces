This is a 3-D version of the mandelbrot set using the Three.js library (http://threejs.org/). The demo requires a modern browser with WebGL support and a graphics accellerator card. During my tests, Google Chrome 28+ performed well, Firefox 23+ worked a little slower and Safari 5.1.7 required patience. Internet Explorer won't work until version 11. iOS devices don't work either (they support WebGL only for advertisement).
